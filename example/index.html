Shadowllax

A MAD simple and ULTRA light parallax library that requires no js initialization code.

NOTE: THIS DEMO IS NOT OPTIMIZED FOR MOBILE DEVICES. Responsive web design is boring and I didn't want to use any external libraries. ¯\_(ツ)_/¯

You can create a customizable parallax with a few HTML classes. Extra CSS can be used to style the content. Any default styles can be easily overriden with your own CSS if you want to.
This is an image used as a background with just the `width` CSS property set to `100%`.

Image by skeeze from Pixabay

This part is the "above-parallax" class. It has a slight shadow to show that it is slightly raised.
The default look is an opaque dark background but that can be easily overriden.
Normal CSS properties work just fine and you can override anything. Glass effect is made with opacity CSS property.

Multiple Layers

You can add multiple layers to the same parallax. The layers need not be an image, it can be any html element.
Each layer can be at a different "depth" ie move at different speeds.

To use the library, first include the css(in the head) and js (At the end of the body)
