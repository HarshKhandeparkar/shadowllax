Shadowllax

A MAD simple and ULTRA light parallax library that requires no js initialization code.

This is the "parallax-content" class ie content at the level of the rest of the content on the page. This is an image used as a background without ANY extra js or css. Although you need to decide the color of this text with your own css.

Image by skeeze from Pixabay

This part is the "above-parallax" class. It has a slight shadow to show that it is slightly raised.
Normal CSS properties work just fine and you can override anything. Glass effect is made with opacity CSS property.

Multiple Layers

You can add multiple layers to the same parallax. The layers need not be an image, it can be any html element.
Each layer can be at a different "depth" ie move at different speeds.

Layers other than the ones inside the parallax will have transparent background by default. Change the background color so that it is visible when the parallax layer moves under it. Non colored backgrounds can be used to make glass like effect.
